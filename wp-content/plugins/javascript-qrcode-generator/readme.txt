== Description ==

QR stands for Quick Response Codes and was developed by Toyota subsidiary Denso-Wave in 1994 to help remove problems in tracking automotive parts. QR code are a type of two-dimensional barcode that can be read using SmartPhone and dedicated QR reading devices, that link directly to text, emails, websites, phone numbers and more.

Use this plugin to add QR code in wordpress post and sidebar widget. This plugin use [JavaScript library](http://davidshimjs.github.io/qrcodejs/ "qrcodejs") by **davidshimjs** to generate QR code. You can add QR code in WP post using shortcode 
[jsqr msg="your text for qr code"  /]. 

Check our online [QR code generator](http://99webtools.com/qrcode.php "QR code generator")

= Plugin Features =

* Easy installation and integration
* No php GD required
* No 3rd party API required
* No server load to generate and display QR code
* Customizable image size and ECC level
* Cross Browser capability
* Sidebar widget
* Custom Wordpress editor button to insert shortcode


== Installation ==

= Using the Plugin Manager =

1. Click Plugins
2. Click Add New
3. Search for `javascript-qrcode-generator`
4. Click Install
5. Click Install Now
6. Click Activate Plugin

= Manually =

1. Upload `javascript-qrcode-generator` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
